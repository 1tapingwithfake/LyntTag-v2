# LyntTag-v2
Dynamic Clan-Tag in Game

![Screenshot](https://i.imgur.com/LUErBlv.gif)

How to configure?

1. Open Init.cpp
2. Go to "setList()" function
3. Edit tags

* For example:
- Tags.push_back("Your tag");
- Tags.push_back("Your tag1");
- Tags.push_back("Your tag2");
- Tags.push_back("Your tag3");
